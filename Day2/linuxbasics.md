**cd** = change directory
**ls** = list
**ls -laR** = list recursively with hidden files.

**mkdir** = make folder
**rmfir** = remove folder
**touch** = create file
**rm** = remove file
**rm -rf** = remove files or folder recursively

**cat** = read content
**less**
**head** = read top lines
**tail** = read bottom lines

**grep** = search pattern
**|**

**ps** = shows process
**kill** = kill process

**chmod** = set permission
**chown** = chane owner
